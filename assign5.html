CSS: class selector

The .class selector selects elements with a specific class attribute.
To select elements with a specific class, write a period (.) character, followed by the name of the class.
This is how it looks in a HTML document:
<​h1 class="example">
And this is how it looks in CSS:
.example { color: cyan; }

CSS: id selector

The #id selector styles the element with the specified id.
To select elements with a specific id, write a period (#) character, followed by the name of the id.
This is how it looks in HTML:
<​h1 id="header">
And this is how it looks in CSS:
#example { color: cyan; }

Floating

The float property is used for positioning and layout on web pages. It can have one of the following values:

  • left - The element floats to the left
  • right - The element floats to the right
  • none - The element does not float. (Default)
  • inherit - The element inherits the float value of its parent

Example: float to the right
This is how it looks in HTML: <​img id="float" src="example.jpg" alt="Example" style="width:50px;height:50px">
And this in CSS: #float { float: right; }
It is also possible to wrap a text with an image.

The clear property lets specifies what elements can float beside the cleared element and on which side. It can have one of the following values:

  • none - Allows floating elements on both sides. (Default)
  • left - No floating elements allowed on the left side
  • right - No floating elements allowed on the right side
  • both - No floating elements allowed on either the left or the right side
  • inherit - The element inherits the clear value of its parent

The most common way to use the clear property is after you have used a float property on an element.
If an element is floated to the left, then you should clear to the left. Your floated element will continue to float, but the cleared element will appear below it on the web page.
